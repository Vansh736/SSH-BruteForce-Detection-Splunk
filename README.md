# SSH Brute Force Detection using Splunk

This project demonstrates how to simulate and detect SSH brute force attacks using Hydra and Splunk SIEM.

## Tools Used

- Kali Linux
- Ubuntu Server
- Splunk Enterprise
- Hydra

## Attack Simulation

Hydra was used to perform brute force attack against SSH service.

## Detection

Splunk was used to analyze authentication logs and detect attacker IP.

## Result

The attacker IP was successfully detected and blocked using firewall rules.
